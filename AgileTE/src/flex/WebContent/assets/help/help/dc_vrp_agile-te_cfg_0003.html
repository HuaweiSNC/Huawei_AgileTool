操作步骤

  1. 获取软件安装包

    1. 登录网站http://support.huawei.com，进入“软件中心 > 版本软件 > 网络OSS与服 务 > iManager U2000 > iManager U2000 > iManager U2000 > iManager U2000 V100R009”。
    2. 选择需要下载的版本
    3. 下载保存在本地
  2. 安装
    1. 双击AgileTE1.0.exe安装包程序开始安装。双击后系统会弹出“选择安装语言”对话框，按照自身需求选择后点击“确定”。
    2. 系统会弹出“安装程序”对话框，单击“下一步”。
    3. 系统会出现“选择安装文件夹”对话框，点击“浏览”选择安装的目的文件夹。选择好之后，单击“下一步”。
    4. 系统会弹出
  3. 修改IP地址
    1. 进入Agile TE的安装根目录。
    2. 修改 startDeploy.bat 下 ops_ip 、 tomcat_ip 为本机 Ip 地址，进入 \tomcat7\webapps\AgileTeService\conf 下 my.ini 中修改 SimulantData = -2 。
    3.  
  4. （可选）修改端口号

    OPS2.0默认绑定的是8080端口，如果8080端口已被占用，请修改其它的端口, 具体请找到安装目录下面的 OPS2.0.bat 文件，将 set ops_port=8080 修改为其它值。如果要对外网开放访问，修改 OPS2.0.bat 文件，将 set ops_ip=localhost 修改为具体的 IP 地址。

    1. 进入Agile TE的安装根目录。
    2. 用“记事本”程序打开OPS2.0.bat 文件，将set ops_port=8080修改为其它值。
