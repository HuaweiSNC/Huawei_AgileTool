进入Agile TE服务

本节包括新建服务器、服务列表以及进入服务。

操作步骤

  1. 新建服务及服务列表
    1. 点击页面侧边栏“服务器配置”进入服务器配置页面。
    2. 点击“新建”弹出“注册服务器”对话框。

    3. 按照如下说明填写文服务器信息。

      表1 填写对象及取值说明

      填写对象

      取值说明

      服务器名称

      字符串形式，可以是中文或者英文。长度范围为0～64。

      服务器IP

      点分十进制格式。

      服务器端口

      整数形式，取值范围为1～65535。

      连接方式
      • http
      • https

    4. 填写完毕后，点击“确定”提交配置。

    说明：

    可以多次执行上述步骤，添加多个服务器。每次添加成功的服务器都将存储在服务列表中。这其中可以有一个作为主服务器，其他的作为备用服务器。

  2. 点击“选择主服务器”，在滑出的列表中选择主服务器。
  3. 点击“进入主服务器”，将进入“General View”页面。
